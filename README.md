# 基于Nuxt3 + TailwindCSS + ElementPlus 的快速开发模板 

> 1. 查看[Nuxt3官方文档](https://nuxt.com/)
> 2. 查看[Nuxt3中文文档](https://nuxt.com.cn/)
> 3. 查看[Nuxt3翻译文档](https://ezdoc.cn/docs/nuxtjs/)

## 安装依赖
node版本：18.18.0
``` bash 
pnpm i 
```

## 开发环境要求
本地node运行版本：18.18.0

### 本地启动命令
``` bash
nuxt dev
```
## 打包环境要求
打包node运行版本：18.18.0

### 本地预览命令
``` bash
nuxt preview
```
### 打包部署命令
线上版本
``` bash
nuxt build
```
